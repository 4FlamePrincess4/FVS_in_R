This subdirectory contains examples of how to implement scripts in the parent directory for five different types of runs: 
1) No treatment or wildfire
2) Wildfire without treatment
3) Mechanical treatment without wildfire (same workflow for rxfire with a different KCP)
4) Mechanical treatment with wildfire
5) Two wildfires in one scenario

These example scenarios may be repurposed for other types of FVS scenarios. 
You may wish to use different keywords and output different tables, but these examples will help you get started with running FVS in parallel in R.
